# vue-cli4-ssr
根据 github 上同名项目修改 增加异步请求

## Project setup
```
npm install
```

### SSR开发模式支持热加载，避免服务端调试重复Build
```
npm run dev:ssr
```

### SPA开发模式
```
npm run dev:spa
```

### 打包发布
```
npm run build:ssr

npm run start
```
